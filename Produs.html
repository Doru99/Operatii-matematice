Observaţii
Produsul AB a două matrici nu se poate efectua daca numărul de coloane ale lui A este egal cu numărul de linii ale lui B. Dacă matricele sunt pătratice atunci are sens întotdeauna atât AB, cât şi BA.
Produsul a 2 matrici
Fie A = (ak,i) ∈ Mm,n (C), B = (bi,j) ∈ Mn,p (C). Produsul dintre A şi B este egal cu matricea C = (ck,j)
Asociativitatea înmulţirii.
Înmulţirea matricilor este asociativă, adică

(AB)C = A(BC)
Distributivitatea înmulţirii.
Înmulţirea matricilor este distributivă, adică

(A + B) C = AC + BC
Elementul neutru
Fie In ∈ Mn (C), unde toate elementele de pe diagonala principală sunt 1, iar restul 0, numită matricea unitate, atunci înmulţirea matricilor admite ca element neutru In

InA = AIn = A
