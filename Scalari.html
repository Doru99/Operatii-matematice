Înmulţirea cu scalari a matricelor
Fie α ∈ C şi A = (ai,j) ∈ Mm,n (C). Se numeşte produsul dintre scalarul α şi matricea A matricea αA obţinută prin înmulţirea elementelor matricei A cu α:
Comutativitatea
Fie α,β ∈ C şi A = (ai,j) ∈ Mm,n (C):

(αβ) A = α (βA)
Distributivitatea scalarului
Fie α ∈ C şi A = (ai,j), B = (bi,j) ∈ Mm,n (C):

α (A + B) = αA + αB
Distributivitatea matricei
Fie α,β ∈ C şi A = (ai,j) ∈ Mm,n (C):

(α + β) A = Aα + Aβ
Elementul neutru
Fie A = (ai,j) ∈ Mm,n (C). Înmulţirea cu matrici admite ca elemnt neutru e=1 :

A * 1 = A
