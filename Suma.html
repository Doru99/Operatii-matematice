Adunarea matricilor se realizează astfel:
Asociativitate
Adunarea matricilor este asociativă, adică:

(A + B) + C = A + (B + C)
Comutativitate
Adunarea matricilor este comutativă, adică:

A + B = B + A
Element neutru
Adunarea matricilor admite matricea nulă ca element neutru. Matricea cu toate elementele egale cu 0 se numeşte matrice nulă:

A + Om,n = A
Elemente simetrizabile
Orice matrice A are un opus notat -A, astfel încât:

A + (-A) = Om,n
